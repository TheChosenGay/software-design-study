## 关于本站

这是一个软件设计能力提升网站，通过刻意练习帮助你掌握：
- **系统设计**：架构设计、分布式系统
- **存储**：Redis、Kafka、MySQL
- **编码设计**：代码质量、设计模式
- **基础设施**：Docker、K8s
- **AI**：LLM、RAG

## 学习方式

1. 说"开始软件测试"启动考试
2. 回答问题后获取详细反馈
3. 薄弱点会记录到对应专题页面
4. 在网站上查看完整答案和学习资料
